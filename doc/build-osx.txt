Mac OS X bitcoind build instructions
Laszlo Hanyecz <solar@heliacal.net>
Douglas Huff <dhuff@jrbobdobbs.org>

Modified for LoMoCoin by Jeff Larkin <jefflarkin@gmail.com>


See readme-qt.rst for instructions on building LoMoCoin QT, the
graphical user interface.

Tested on 10.5 and 10.6 intel.  PPC is not supported because it's big-endian.

All of the commands should be executed in Terminal.app.. it's in
/Applications/Utilities

You need to install XCode with all the options checked so that the compiler and
everything is available in /usr not just /Developer I think it comes on the DVD
but you can get the current version from http://developer.apple.com


1.  Clone the github tree to get the source code:

git clone git@github.com:lomocoin/lomocoin.git lomocoin

2.  Download and install MacPorts from http://www.macports.org/

2a. (for maximum compatibility with 32-bit installs)
    Edit /opt/local/etc/macports/macports.conf and uncomment "build_arch i386"

    sudo nvram boot-args="arch=x86_64"
    sudo nvram boot-args="arch=x86_32"

3.  Install dependencies from MacPorts

sudo port install boost db48 openssl miniupnpc qt4-mac qrencode

# 如果使用brew安装,请使用 brew install berkeley-db(db48)
brew install automake berkeley-db4 libtool boost  miniupnpc openssl pkg-config protobuf  qt5 libevent

Optionally install qrencode (and set USE_QRCODE=1):
sudo port install qrencode

4.  Now you should be able to build bitcoind:

cd lomocoin/src
make -f makefile.osx

# To build 64-bit binaies:
make -f makefile.osx RELEASE=true 64BIT=true

5.make Makefile
cd lomocoin
#change path
vim bitcoin-qt.pro
qmake bitcoin-qt.pro

6. make gui
make

# clean  : make clean
7 make dmg
 macdeployqt lomocoin-Qt.app    -verbose=1 -dmg

Run:
  ./lomocoind --help  # for a list of command-line options.
Run
  ./lomocoind -daemon # to start the bitcoin daemon.
Run
  ./lomocoind help # When the daemon is running, to get a list of RPC commands
